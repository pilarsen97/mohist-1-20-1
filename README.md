# Сервер mohist 1.20.1

## Запуск
1. Скачайте или склонируйте репозиторий.
> `git clone https://github.com/pilarsen97/mohist-1-20-1`
2. Откройте файл [server.properties](https://github.com/pilarsen97/mohist-1-20-1/blob/main/server.properties).
3. Измените строку `server-ip=` на ваш ip.
4. Сохраните файл [server.properties](https://github.com/pilarsen97/mohist-1-20-1/blob/main/server.properties).

#### Для Windows:
5. Запустите файл [launch.bat](https://github.com/pilarsen97/mohist-1-20-1/blob/main/launch.bat).
6. Ожидайте сообщения **Done** в консоли.
7. Подключайтесь к серверу через ваш ip.

#### Для других OS:
5. Запустите файл [launch.sh](https://github.com/pilarsen97/mohist-1-20-1/blob/main/launch.sh).
6. Ожидайте сообщения **Done** в консоли.
7. Подключайтесь к серверу через ваш ip.
